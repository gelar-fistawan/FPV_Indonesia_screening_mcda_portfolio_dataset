# Methodology summary

This package supports an uncertainty-aware national screening and portfolio planning workflow for floating photovoltaic (FPV) deployment on Indonesian reservoirs and inland water bodies.

Workflow:
1. Candidate inventory construction from reservoir and inland-water datasets.
2. Stable-water screening using surface-water persistence evidence.
3. Criteria extraction and normalization into 0–1 benefit scores.
4. Objective MCDA using equal weights, Entropy, CRITIC, and TOPSIS.
5. Monte Carlo rank acceptability analysis using 3,000 sampled weight vectors.
6. Preliminary portfolio optimization for 1, 3, 5, and 10 GW planning scenarios.

The outputs are intended as screening and planning-support artifacts, not final deployment recommendations.
