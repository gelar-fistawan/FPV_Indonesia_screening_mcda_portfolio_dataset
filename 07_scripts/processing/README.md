# Processing scripts

This folder documents the processing workflow used to generate the final criteria matrix and MCDA-ready datasets. The research dataset deposited here contains processed outputs and analysis results, not local file-management scripts.

Recommended reproducibility sequence:
1. Build and screen candidate inventory.
2. Extract JRC stable-water and criteria variables.
3. Construct final criteria matrix.
4. Run objective weighting, TOPSIS, rank acceptability, and portfolio scenarios.
