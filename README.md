# FPV Indonesia screening, MCDA, and portfolio planning dataset

This repository contains data products, processed matrices, analysis outputs, JPEG figures, manuscript tables, and reproducibility materials supporting a study on uncertainty-aware floating photovoltaic (FPV) portfolio planning for Indonesian reservoirs and inland water bodies.

The repository does **not** include manuscript drafts or publisher-formatted article files. The manuscript is handled separately through the journal submission process.

## Contents

- `01_source_data_documentation/`: notes on external source datasets.
- `02_processed_data/`: final criteria matrix and processed screening outputs.
- `03_mcda_results/`: MCDA rankings, weights, and method comparison files.
- `04_uncertainty_portfolio/`: rank acceptability and portfolio scenario outputs.
- `05_manuscript_tables/`: manuscript-ready table workbook.
- `06_figures/jpeg/`: JPEG manuscript figures only.
- `07_scripts/`: documentation and available research scripts.
- `08_qc_logs/`: name-verification and limitation notes.

## Important interpretation limits

These outputs are screening and planning-support materials. They should not be interpreted as site-level engineering designs, permitting decisions, grid interconnection approvals, or investment recommendations.
