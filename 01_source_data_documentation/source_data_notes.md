# Source data documentation

This package includes processed matrices and analysis outputs. It does not redistribute restricted or externally licensed source datasets unless already present as derived outputs. Users should obtain original datasets from their official providers and cite those providers.

Core external sources:
- GRanD reservoir database
- HydroLAKES
- JRC Global Surface Water
- ERA5-Land
- WDPA
- WRI Global Power Plant Database

The processed outputs are intended for transparency and reproducibility of the screening, MCDA, uncertainty, and portfolio scenario results.
